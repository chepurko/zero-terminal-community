# Zero Terminal Community

This is the public feedback hub for [Zero Terminal](https://zeroterminal.io) — a portfolio intelligence platform for active traders.

## How to participate

- 🐞 **Bug Reports** — something broken? [Open a discussion](https://github.com/chepurko/zero-terminal-community/discussions/categories/bug-reports).
- 💡 **Feature Requests** — [tell us what to build next](https://github.com/chepurko/zero-terminal-community/discussions/categories/feature-requests). Upvote existing requests.
- ❓ **Questions** — [ask anything](https://github.com/chepurko/zero-terminal-community/discussions/categories/questions) about using Zero Terminal.
